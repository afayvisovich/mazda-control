# 🔧 Исправление сервера подключения v4.2

## 🎯 Проблема

В версии v4.0 использовался сервер **`172.16.2.30:50001`** (AG35TspClient), но он **НЕДОСТУПЕН**.

### Доказательства из логов:

1. **Системный AG35TspClient** (PID 6624) постоянно получает ошибки:
   ```
   03-13 21:51:08.011  6624  6833 E AG35TspClient: IOException
   03-13 21:51:14.066  6624  6768 I AG35TspClient: no connect socket!
   ```

2. **Штатное приложение автомобиля** использует **`localhost:32960`**:
   ```
   03-11 22:13:35.851  8755  8887 I CallServerInterceptor: 
   request <3 ip = localhost/127.0.0.1 port = 32960 >
   ```

3. **Все команды окон/спойлера** идут через порт 32960 с успешными ответами.

---

## ✅ Решение

**ИСПОЛЬЗУЕМ ПРАВИЛЬНЫЙ СЕРВЕР: `127.0.0.1:32960` (Fake32960Server)**

### Изменения в v4.2:

| Файл | Изменение |
|------|-----------|
| `TBoxSpoilerController.kt` | Сервер изменён на `127.0.0.1:32960` |
| `MainActivity.kt` | Диагностика проверяет порт 32960 |
| `network_security_config.xml` | Приоритет localhost |
| `NETWORK_DIAGNOSTICS.md` | Обновлена документация |
| `AG35TSPCLIENT_MIGRATION.md` | Исправлено описание |

---

## 📋 Почему это правильно?

### Порт 32960 (localhost/Fake32960Server):
- ✅ **РАБОТАЕТ** - все оригинальные логи показывают успешное взаимодействие
- ✅ Используется штатным приложением автомобиля
- ✅ Все команды окон/спойлера идут через этот порт
- ✅ Работает **локально** (не требует сети)

### Порт 50001 (172.16.2.30/AG35TspClient):
- ❌ **НЕ РАБОТАЕТ** - постоянно `IOException`
- ❌ Системный AG35TspClient не может подключиться
- ❌ Это **другой сервер**, который недоступен в вашем автомобиле

### 1. Запустите приложение

- Автомобиль выполнит команду

## 📊 Технические детали

### Протокол:
- 14-байтовый заголовок (magic byte `0x5A`)
- Тело пакета (512 байт)
- CRC16 checksum (2 байта)
- **Итого:** 548 байт

### Команды:
**ОТКРЫТЬ:**
- Function Bytes: `BF 11 0D 06 27 21 20 01`
- Byte 355: `0x2B`
- Byte 363: `0x5A`

**ЗАКРЫТЬ:**
- Function Bytes: `BF 11 0D 05 27 23 20 01`
- Byte 355: `0x2B`
- Byte 363: `0x5A`

### Время движения:
- ~2.2 секунды на полное открытие/закрытие

## ❓ Вопросы

### Почему не работает 172.16.2.30:50001?
Этот сервер может быть недоступен в вашей конфигурации автомобиля или требует специальных разрешений.

### Будет ли работать на других автомобилях?
Возможно, потребуется изменить сервер на правильный для вашей модели. Используйте диагностику для проверки.

### Можно ли использовать оба сервера?
Да, можно добавить переключение между серверами в настройках.
